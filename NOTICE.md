# Notice

This repository contains the source code and documentation for a Chrome extension UI prototype.

## Project nature

This project is a **demonstration UI prototype only**. It does not analyze, scrape, transmit, or otherwise process content from any website. All report text shown in the popup is hard-coded sample data.

## Third-party dependencies

No third-party package dependencies are committed in the current repository.

## Assets

The extension icons are generated from `icons/icon16.svg`, an original SVG asset in this repository. If the icons are replaced later, only use assets that are original, public-domain, or licensed for the intended open-source and distribution use.

## Trademarks

Earlier development drafts referenced several third-party video platforms by name. Those references have been removed. This project is not affiliated with, endorsed by, or sponsored by any third party. Any incidental textual or visual similarity is unintentional; please file an issue if you spot a remaining reference.

## User and platform content

Do not commit captured videos, subtitles, comments, exports, screenshots, or other platform/user content to this repository. The prototype is not designed to handle such content and accepting it via pull requests is out of scope.
