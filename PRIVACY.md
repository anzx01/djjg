# 隐私说明

生效日期：2026-05-23（最新修订：UI 原型化重构）

本项目是一个 Chrome 扩展的 **UI 原型**，不收集、不传输、不存储任何用户数据。

## 扩展会访问的数据

无。

- 扩展未声明任何 `host_permissions`
- 扩展未注入任何 `content_scripts`
- 扩展未注册 `background` 脚本
- 弹窗页面（popup）不发起任何网络请求
- 弹窗内显示的报告内容为脚本中预设的静态示例文本，与任何实际网页内容无关

## 权限

`manifest.json` 中目前仅保留 `storage` 权限的占位声明，当前版本未实际写入数据。如果未来用到，将在本说明中明确写入哪些字段、保留多久、如何清除。

## 数据存储和传输

- 扩展不读取、不缓存任何外部内容
- 扩展不向项目作者、第三方服务或分析平台传输任何数据
- 卸载扩展即可移除全部本地痕迹

## 历史版本说明

更早的开发草稿（已从仓库中移除）曾尝试读取浏览页面的标题与文案。当前发布版本中所有相关代码均已删除。

## 未来变更

如果未来版本接入任何外部接口、AI 服务、账号系统或数据同步，发布前必须更新本说明和 README，并在产品界面中清楚告知数据使用方式。

## 联系

请通过 GitHub Issues 反馈隐私问题。不要在公开 Issue 中粘贴密钥、账号凭据或未经授权的第三方内容。
