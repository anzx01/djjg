# 隐私说明

生效日期：2026-05-23

短剧解构师是一个本地运行的 Chrome 扩展。当前版本不会把页面内容、账号信息或分析结果上传到远程服务器，也不会调用外部 AI API。

## 扩展会访问的数据

在用户访问受支持的平台页面时，扩展可能读取以下页面可见信息，用于识别页面类型和生成本地分析：

- 当前标签页的 URL、标题和平台类型。
- 页面中公开显示的视频标题、描述、字幕/文案、作者名、播放量、点赞数、弹幕数、发布时间和剧集信息。
- 用户主动点击分析按钮后产生的本地分析结果。

扩展不会主动读取或收集登录密码、支付信息、Cookie、私信、通讯录或浏览器外部文件。

## 数据存储和传输

- 当前版本的数据保存在 `chrome.storage.local` 或扩展运行内存中。
- 当前版本不向项目作者、第三方服务或分析平台传输用户数据。
- 卸载扩展，或在 Chrome 扩展管理页清除该扩展的数据，可以删除本地保存的数据。

## 第三方内容

扩展只用于分析用户有权访问的页面内容。页面中的视频、音频、字幕、图片、标题和评论等内容可能受平台规则和版权法保护。用户不得使用本扩展复制、传播、下载或改编无授权内容。

## 未来变更

如果未来版本接入云端 AI、账号系统、数据同步或任何外部接口，应在发布前更新本说明和 README，并在产品界面中清楚告知数据使用方式。

## 联系

请通过 GitHub Issues 反馈隐私问题。不要在公开 Issue 中粘贴密钥、账号凭据或未经授权的第三方内容。
