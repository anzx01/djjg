# 短剧解构师 Chrome插件

> 一键AI解构短剧：提炼摘要、标注爽点、分析套路，为创意改编提供蓝图

## 项目简介

短剧解构师是一个专为短视频创作者和内容分析师设计的Chrome浏览器插件。通过AI技术自动分析抖音、西瓜视频、快手、B站等平台的短剧内容，提取关键信息，为内容创作和改编提供数据支持。

> 当前版本为测试版，分析结果为本地模拟生成；不会上传页面内容，也不会调用外部AI服务。

## 核心功能

### 🎯 智能分析
- **剧情摘要**：自动提炼短剧核心剧情和关键情节
- **爽点标注**：识别并标记观众喜爱的精彩片段
- **套路分析**：分析常见的剧情模式和创作套路
- **改编建议**：基于分析结果提供创意改编建议

### 📊 数据提取
- 视频标题和描述
- 播放量、点赞数、评论数
- 作者信息和标签
- 发布时间和互动数据

### 🎨 平台支持
- ✅ 抖音 (douyin.com)
- ✅ 西瓜视频 (ixigua.com)
- ✅ 快手 (kuaishou.com)
- ✅ B站 (bilibili.com)
- 🔄 更多平台持续支持中...

## 安装方法

### 开发模式安装
1. 克隆或下载项目到本地
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

### 图标设置
按照 `icons/README.md` 的说明创建PNG图标文件；如需替换，请使用自有或已获授权的图标素材。

## 使用方法

### 基本使用
1. 访问支持的短剧平台（抖音、西瓜视频、快手、B站）
2. 打开任意短剧播放页面
3. 点击浏览器工具栏中的插件图标
4. 点击"开始解构当前短剧"按钮
5. 等待分析完成，查看结果

### 查看历史记录
- 分析结果会自动保存到本地存储
- 可在插件设置中查看历史分析记录
